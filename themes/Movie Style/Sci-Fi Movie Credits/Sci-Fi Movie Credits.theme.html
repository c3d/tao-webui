[[caption]]

[[caption]] is a collection of pages to create a simple movie credits animation similar to the Star Wars intro, using the Movie Credits module. This collection features five different configurable screens.

Introduction screen

The introduction screen shows your message flying towards you, in large letters with a green outline.

Location screen

The location screen shows a little bit of context on the presentation. You can use it to say where you are presenting and who you are presenting to.

Crawling text screen

The crawling text screen shows a relatively long text that scrolls towards the top of the screen.

Crawling text screen with Ships

This is similar to the previous one, but adding a pair of flying ships.

Final credits screen

The final credits screen screen is where you can put thank-you notes and mention who helped you with the presentation.

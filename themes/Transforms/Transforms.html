Transforms

This category contains a number of geometrical transforms that can apply to any 2D or 3D shape.

  • Translations move objects around.
  • Scaling change the size of objects.
  • Rotations spin objects.

Axes

Initially, the X axis is the horizontal axis, positive values towards the right. The Y axis is the vertical axis, positive values towards the top. The Z axis is the depth axis, positive values towards the viewer.

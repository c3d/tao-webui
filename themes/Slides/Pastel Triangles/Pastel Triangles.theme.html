[[caption]]

[[caption]] is is a dynamically animated slide theme featuring smooth transitions between automatically generated backgrounds.

The dynamic backgrounds use triangles tinted in random pastel colors. There is a smooth transition between pages, as well as a transition at regular intervals. As a result, the page background never look exactly the same twice.

You can use this theme directly, or as a starting point for your own slide designs.

[[include "/themes/Slides/SlideTheme.theme.html"]]

Implementation

The theme [[caption]] is implemented in the Tao application directory under modules/pastel_triangles.
